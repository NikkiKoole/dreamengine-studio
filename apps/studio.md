## About

Small, honest tools made by one person. The work is built on a home-grown fantasy
console — deep, real simulations behind a humble lo-fi surface — where each release is
one focused idea done properly rather than a feature list. No dark patterns, no ads,
no data harvesting; you buy the thing, you own the thing.
